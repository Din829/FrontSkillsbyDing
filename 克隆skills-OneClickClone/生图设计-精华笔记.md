## 组合变化引擎（防止生图千篇一律）

来源：taste-skill 的 `imagegen-frontend-web` 第2节。用途：让生图**每次不一样**，不收敛到同一种
SaaS 模板。**用法定位**：这是给"写生图意图的主 Agent"挑选用的——主 Agent 从每个维度选一个、
组合成一句设计意图，传给 `gen-refs.cjs` 的 `<设计意图>` 参数。**不要塞进 gen-refs 的脚本 prompt**
（那会变回硬指示，违背软指示哲学）。

各维度选一个，组合执行（别全堆一起）：

- **主题 Theme**：① Pristine Light（米白/纸感+利落深字）② Deep Dark（炭灰/石墨，必要时才发光）
  ③ Bold Studio Solid（强控色块：酒红/宝蓝/森绿/朱红/翡翠 + 高对比 UI）④ Quiet Premium Neutral
  （骨白/沙/灰褐/烟，克制奢华）
- **背景 Background**：① 细技术网格/点阵 ② 纯色场+柔和氛围渐变 ③ 全幅电影级图像（控好对比）
  ④ 安静纹理纸/材质/触感面
- **字体 Typography**：① Satoshi 类干净 grotesk ② Neue Montreal 类精炼 grotesk ③ Cabinet/Clash 类
  表现型 display ④ Monument 类压缩声明体 ⑤ 优雅 serif+sans 搭配 ⑥ Swiss 理性 sans 强层级。
  （别飘回无聊的默认 web 字体）
- **Hero 架构**：① 电影级居中极简 ② 非对称分栏 ③ 漂浮拍立得散布 ④ 内联巨型字 ⑤ 编辑式偏置构图
  ⑥ 图像优先大 Hero（文字克制）
- **区块系统**（选 1 主结构）：① 严格模块 bento ② 交替编辑块 ③ 海报式堆叠叙事 ④ 画廊式视觉节奏
  ⑤ Swiss 网格纪律 ⑥ 非对称高级营销流
- **签名组件**（恰好选 4 个）：斜向错落方块 masonry / 3D 层叠卡组 / 悬停手风琴切片 / 无缝 bento /
  无限品牌跑马灯 / 旋转拍立得弧 / 垂直节奏线 / 离格编辑布局 / 产品 UI 面板堆 / 分栏证言墙 /
  超大指标条 / 分层图像裁切框
- **动效语言**（恰好选 2 个）：文字擦除 / 钉住叙事 / 错落浮起 / 视差图像漂移 / 平滑手风琴 / 电影淡入
- **构图锚点**：左文右图允许，但它是**最被滥用的 AI 套路**——别当默认，只在真最合适时才用。

> 例：主 Agent 可拼出意图 = "Bold Studio Solid（森绿）主题 + Swiss 网格 + 非对称分栏 Hero +
> 超大指标条/无缝bento/分层图像裁切/产品UI面板堆 + 错落浮起动效"，传给 gen-refs。
